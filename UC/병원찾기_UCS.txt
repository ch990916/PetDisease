목표 : user는 원하는 위치를 중심으로 근처에 있는 동물병원들의 위치를 알 수 있다.

성공 시나리오
	1. user는 자신의 위치 정보 제공 팝업에 동의한다.
	2. 시스템은 user의 위치 정보를 중심으로 주변 지도와 로드뷰를 제공한다.
	3. 시스템은 지도의 중심을 기준으로 반경 3 km안 동물병원들의 이름 / 주소를 제공한다.
	4. user는 지도에 표시하고 싶은 동물병원의 정보를 클릭한다.
	5. 시스템은 user가 선택한 동물병원의 위치를 중심으로 주변 지도와 로드뷰를 제공한다.
확장 시나리오
	1.1 user가 위치 정보 제공 팝업에 동의하지 않을 경우:
		1.1.1 시스템은 user가 다른 위치 정보를 입력하기 전까지 지도와 로드뷰를 사용 불가하다고 알려준다.
	2.1 user가 위치 입력창에 자신이 원하는 위치 정보를 입력할 경우:
		2.1.1 시스템은 user가 입력한 위치 정보를 중심으로 지도와 로드뷰를 제공한다.
	3.1 반경 3 km내에 동물병원이 없을 경우:
		3.1.1 시스템은 동물병원 정보 란에 주위에 동물병원이 없다는 메세지를 제공한다.
