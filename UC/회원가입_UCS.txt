목표 : user는 회원가입을 통해 자신의 정보를 DB에 저장하고 member가 될 수 있다.

성공 시나리오
	1. user는 닉네임 입력칸에 원하는 닉네임을 입력한다.
	2. user는 주민번호 입력칸에 자신의 주민번호 앞자리 7자리를 입력한다.
	3. user는 id 입력칸에 원하는 아이디를 입력한다.
	4. user는 pw 입력칸에 원하는 비밀번호를 입력한다.
	5. user는 pw 확인칸에 pw 입력칸에서 입력했던 pw를 똑같이 입력한다.
	6. user는 이메일 입력 칸에 자신의 이메일을 입력한다.
	7. user는 우편번호나 주소칸을 클릭하면 나오는 팝업창에서 자신의 주소를 검색한다.
	8. user는 검색한 뒤 나오는 주소들 중 자신의 주소를 선택한다.
	9. 시스템은 user가 선택한 주소 정보에 맞춰 우편번호와 주소를 표시해준다.
	10. user는 상세 주소란에 자신의 상세 주소를 입력한다.
	11. user는 파일 선택 버튼을 누르고 자신의 프로필 사진 파일을 선택한다.
	12. user는 정보를 다 입력한 뒤 가입 버튼을 누른다.
	13. 시스템은 입력 정보들을 DB에 저장한다.
	14. 시스템은 가입 성공과 메인 페이지에서 로그인을 요청하는 메세지를 띄운다.
	15. 시스템은 user를 메인 페이지로 보낸다.
확장 시나리오
	1.1 입력한 닉네임을 이미 다른 member가 사용하고 있을 경우:
		1.1.1 시스템은 user가 입력한 닉네임의 글자색을 red로 바꾼다.
	3.1 입력한 id를 이미 다른 member가 사용하고 있을 경우:
		3.1.1 시스템은 user가 입력한 id의 글자색을 red로 바꾼다.
	12.1 user가 입력한 부분에 문제가 있을 경우:
		12.1.1 시스템은 어느 부분 입력이 잘못되었는지 경고창으로 알려준다.
		12.1.2 시스템은 입력이 잘못된 부분의 입력창을 초기화한다.
		12.1.3 시스템은 입력이 잘못된 부분의 입력창으로 커서를 이동시킨다.
