1. 기본적으로 anconda를 사용중이어서 conda 가상환경에서 진행
	flask 구동에 필요한 파일들이 하나의 폴더 안에 있다고 가정함
	
	ㄱ.	conda create -n pet python=3.10					pet이라는 python 3.10이 적용된 가상환경 생성
	
	ㄴ. conda activate pet								가상환경(pet 활성화)
	
	ㄷ. cd (flask 구동에 필요한 파일 담아둔 폴더)		경로 이동
	
	ㄹ. pip install -r requirements.txt					requirements.txt 안에 있는 lib 설치
			혹시 torch 설치 중 에러가 발생하는 경우
			pip3 install torch torchvision torchaudio		해당 명령어 사용
	
	ㅁ. python skin_Detect.py							skin_Detect flask 실행
