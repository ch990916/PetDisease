멤버 권한 추가 : (pm_admin)
	관리자 권한 글쓰기 로그인
		0       0	      0   으로 생각해주시면 됩니다.

	0: 로그인 불가
	1: 로그인 가능, 글쓰기 불가능
	3: 로그인 가능, 글쓰기 가능
	4: 관리자

관리자 기능(회원 관리) 추가
	멤버 로그인 시 권한에 따라 ${loginPage}의 내용이 바뀝니다.
	일반 권한(1,3) 회원 : 회원정보 / 로그아웃 에 접근 가능
	관리자 권한 회원 : 닉네임 앞에 ADMIN이 추가됩니다(loginPage에서만, 실제 닉네임에는 변동 없음).  회원관리 / 로그아웃 접근 가능

	회원 관리 : 현재 가입된 멤버의 목록이 표시됩니다. 정렬은 권한(내림차순),아이디(오름차순)순 입니다.
		목록에서 아이디를 클릭하면 그 회원의 정보창에 접근 가능합니다.
		정보창에서 회원 탈퇴 및 권한변경이 가능합니다.
